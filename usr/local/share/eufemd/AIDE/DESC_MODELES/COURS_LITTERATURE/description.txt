+ Modèle : COURS_LITTERATURE
+ Activité : cours
+ Matière : Littérature

Sorties générées :

+ Une présentation du cours (diaporama PDF),
+ Une prise de notes complète (PDF papier, A4) 
+ Et une version trouée (idem)

Champs spécifiques :

| Champ        | Valeur                             |
| ------------ | ---------------------------------- |
| TITRE        | Celui de l'œuvre étudiée           |
| OBJET_COURT  | Objet d'étude abrégé               |
| NUM_SEQUENCE | Numéro de la séquence              |
| OBJET_LONG   | Objet d'étude complet              | 
| REF_DESC     | No de réf. ds le descriptif        | 
| ACTIVITE     | Lect. analytique ou complémentaire |

Les champs suivants composent la référence du texte étudié :

| Champ  | Valeur                         |
| ------ | ------------------------------ |
| AUTEUR | Auteur                         |
| TITRE  | Titre de l'œuvre ou du recueil |
| DETAIL | chapitre, titre de poème, etc. |
