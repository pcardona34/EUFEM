+ Modèle : METHODE_LITTERATURE
+ Activité : Méthode
+ Matière : Littérature

Ce modèle est une version dérivée et allégée du modèle de cours en 
littérature.

Sorties générées :

+ Une présentation (diaporama PDF),
+ Une prise de notes complète (PDF papier A4) ;
+ Et une version trouée (idem).

Champs spécifiques :

| Champ        | Valeur                             |
| ------------ | ---------------------------------- |
| TITRE        | Celui de la leçon                  |
| OBJET_COURT  | Objet d'étude abrégé               |
| NUM_SEQUENCE | Numéro de la séquence              |
| OBJET_LONG   | Objet d'étude complet              | 
| ACTIVITE     | Méthode                            |
